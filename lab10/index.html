Lab 10 - JavaScript for the Web

Challenge

The challenge for this lab was to experiment with DOM manipulation by using JavaScript and jQuery to make a button that generates random Lorem Ipsum text when pushed and then display it.

Problems

We did not have many problems with this lab until we got to the bonus task. It took a while to figure out how to get the text user was inputting to appear on the right side of the screen without it looking weird or messing up the format.

Reflection

This lab was useful as it allowed us to experiment with DOM manipulation and learn how to generate random sets of text. It also allowed us to learn how to use jQuery to have users input text as well as how to make inline block elements appear on the right side of the screen.
